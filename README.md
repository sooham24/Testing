# README - Simple Static Form Website

## 📘 Project Title

**Student Information Form Website**

---

## 📖 Description

Welcome to the *Simple Static Form Website* project.
This website has been developed using basic **HTML** and **CSS** technologies to demonstrate the structure and styling of a student form webpage.

As the Principal of this institution, I appreciate the effort put into creating a clean, user-friendly, and responsive form interface. This project reflects the importance of web development fundamentals and encourages students to build strong practical skills in front-end development.

The website includes:

* A student registration form
* Input fields for user details
* Clean layout and styling
* Basic responsive design using CSS

---

## 🛠 Technologies Used

* **HTML5** – For webpage structure
* **CSS3** – For styling and layout

   **OR**
